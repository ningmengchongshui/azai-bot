# Azai-Bot

## Clone

> 必要环境 Windows/Linux + Chrome/Chromium/Edge

> 必要环境 18.18.2>Node.js>16.14.0 + Redis>5.0.0

- 拉取项目

```sh
## github
git clone --depth=1 https://github.com/ningmengchongshui/ayunzai.git
cd ayunzai
```

- 拉取依赖

```sh
npm install pnpm -g
pnpm install
# 外服请先删除.cnpmrc
```

- 机器，启动

```sh
npm run app qq #qq测试机
```

> 如果发现插件解析错误,请删除原插件里的main.js文件
